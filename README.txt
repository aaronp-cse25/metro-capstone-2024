# metro-capstone-2024
CSE Capstone Fall 2024. Group 4 project working with Louisville Metro.

Brief How-to guide:
There is sample output in the 'results' folder. Each file name is self describing.
Please empty the results folder before running. You can store results in a local folder.
Run the file run_scripts to generate output based on your yaml config file.
runtime is 3-5 minutes based on current yaml.
